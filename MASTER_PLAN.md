# Платформа занятости Узбекистана — MASTER PLAN

## Цель
Помочь безработным гражданам Узбекистана найти работу ИЛИ открыть малый бизнес
через умный опрос → AI-рекомендации → пошаговый чеклист действий.

## Целевая аудитория
Безработные граждане Узбекистана, районный уровень, возраст 18–60 лет.
Низкая цифровая грамотность у части аудитории — UI должен быть предельно простым.

## Технический стек
- Frontend: Next.js 14 (App Router) + TypeScript + Tailwind CSS
- Backend: Next.js API Routes
- БД: Supabase (PostgreSQL + RLS)
- AI: Claude API (claude-sonnet-4-5) — streaming
- Карты / конкуренты: 2GIS API
- Деплой: Vercel
- PDF: react-pdf или puppeteer

## Языки интерфейса
Узбекский (латиница) и Русский — переключатель в шапке.

## Два пути пользователя

### Путь A — РАБОТА (ish topish)
Для тех кто хочет найти найм, а не открыть бизнес.
Вопросы: навыки → опыт → готовность к переезду
Результат: список вакансий из hh.uz / mehnat.uz по профилю

### Путь B — БИЗНЕС (biznes ochish)
Для тех кто хочет открыть своё дело.
Вопросы: навыки → капитал → залог → конкуренты → предпочтения
Результат: топ-3 вида бизнеса + банк + бизнес-план + чеклист

---

## Ключевые принципы

1. Опрос максимум 12 вопросов с динамическим ветвлением — не анкета
2. Конкурентный анализ = 2GIS API + самооценка пользователя
3. Рекомендации взвешенные, не категоричные ("возможно", "ориентировочно")
4. Конечный результат = чеклист действий с конкретными ссылками и адресами
5. Трекер прогресса: пользователь отмечает шаги (ИП → кредит → открыл)
6. Истории успеха из того же района — социальное доказательство
7. Дисклеймер об ответственности — обязателен

## Scoring формула (Путь B)
```
score(бизнес) =
  навыки_совпадают   × 0.30 +
  капитал_достаточен × 0.25 +
  конкуренция_низкая × 0.20 +
  риск_в_допуске     × 0.15 +
  сезон_подходит     × 0.10
```

## Подбор банка
```
нет залога         → TBC (без залога до 300 млн), Микрокредитбанк Махалля
есть залог         → все банки
женщина            → приоритет Микрокредитбанк Бизнес-леди
сельский район     → льготное рефинансирование 2026
малообеспеченная   → специальные программы реестра бедных
молодёжь (<30)     → NBU программа молодых предпринимателей
```
